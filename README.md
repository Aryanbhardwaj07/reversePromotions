# reversePromotions


HOTEL PRICE WITH PROMOTIONS

OVERVIEW:
You are given the final selling price of a hotel room after promotions were applied. Your task is to compute the original price before promotions.
PROMOTION PERIOD 
Active on:
    01-Dec to 30-Dec (inclusive)
Applies to:
    All hotels / all hotel rooms
Promotions applied IN THIS ORDER:
    1) Rs. 100 OFF    (fixed discount)
    2) 30% OFF     (percentage discount)
